Upload Attendance Data

Daily workflow: At COB each day, HT Admin (Student) exports the absence report for the day as an Excel file from Sentral.

Sentral path: Sentral -> School Attendance -> Reports -> Absence Lists -> Export

How this upload is processed: the parser checks the Description column and processes any row containing absent or unjustified. It then reads the Time column, splits on -, takes the right-hand side as the arrival time, and compares that time to 08:35.
